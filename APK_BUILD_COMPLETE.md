## 🎯 Quick Start: Install on Phone

### Option 1: USB Cable (Best)
1. Connect phone to PC with USB cable
2. Enable USB Debugging on phone (Settings → Developer Options)
3. Copy APK file from Desktop to phone Downloads folder
4. Open file manager on phone, tap APK file
5. Tap **Install** → **Open**

### Option 2: Email/Cloud (Easy)
1. Email yourself the APK file
2. Open email on phone
3. Download attachment
4. Tap APK to install

### Option 3: ADB Command
```bash
adb install C:\Users\srinath\Desktop\SmartSensrty-RELEASE.apk
```
