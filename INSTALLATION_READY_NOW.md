## 🚀 Quick Installation (Choose 1 Method)

### Method 1: USB Cable (Recommended) ⭐
1. **Connect phone to PC** with USB cable
2. **Enable USB Debugging** on phone:
   - Settings → About Phone → Tap "Build Number" 7 times
   - Settings → Developer Options → USB Debugging ON
3. **Copy APK from Desktop** to phone Downloads folder
4. **Open file manager** on phone → Downloads
5. **Tap APK file** → Install → Open

**Time**: 5-10 minutes

### Method 2: Email Yourself
1. **Email the APK file** to your gmail/email address
2. **Open email on phone**
3. **Download attachment**
4. **Tap APK** to install

**Time**: 2 minutes (depends on network)

### Method 3: Cloud Drive
1. **Upload APK to Google Drive or OneDrive**
2. **Open on phone**
3. **Download and tap** to install

**Time**: 3-5 minutes

## 🆘 Troubleshooting Quick Links

**Problem**: Won't install
- Enable "Unknown Sources" in Settings
- Free up phone storage
- Try different USB port
- Restart phone

**Problem**: Installation blocked
- Clear phone cache
- Try RELEASE version (smaller)
- Update Android OS

**Problem**: App crashes
- Uninstall completely
- Clear cache: Settings → Storage → Clear Cache
- Reinstall fresh

**Problem**: API features not working
- Check Render deployment status
- Use Test API button (🐜) in Emergency History
- Ensure you're logged in
- Check internet connection
